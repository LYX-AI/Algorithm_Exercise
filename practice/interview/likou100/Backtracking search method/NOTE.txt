回溯其实是一个递归
回溯法可以解决哪些问题
    1.组合
    2.切割
    3.子集
    4.排列
    5.棋盘

    回溯法要可以抽象成一个树的简图
    一个节点就是一个搜索组合的大小
    树的深度就是递归的深度
    所有的题解题的时候都要这样想把题目抽象成一个树型结构

    void backtracking(参数)
        #终止条件
        if 终止条件：
           收集结果
           return
        for(集合的元素)：
          处理节点
          递归函数
          回溯
    return
    回溯法
    1.递归函数是没有返回值的一般是void 回溯函数一般起名都叫backtracking(参数)；
    2.关于参数回溯法的参数一般是很多的，不用一开始就想写好而是想到用哪个就之后再补哪个；
    3.要有终止条件的
